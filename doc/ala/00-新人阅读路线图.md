## 快速参考：我想改某个功能去哪找？

| 我想改... | 先看这个文件 | 再看这个目录 |
|-----------|------------|-------------|
| 首页 UI | `happy-app/sources/app/(app)/index.tsx` | `components/SessionsList.tsx` |
| 聊天界面 | `happy-app/sources/app/(app)/session/[id]/` | `components/ChatList.tsx` |
| 新增 API | `happy-server/sources/app/api/routes/` | `prisma/schema.prisma` |
| CLI 命令 | `happy-cli/src/commands/` | `src/index.ts`（参数解析） |
| 加密逻辑 | `happy-app/sources/sync/encryption/` | `happy-cli/src/api/encryption.ts` |
| 添加新语言 | `happy-app/sources/text/translations/` | `text/_all.ts` |
| 数据库表 | `happy-server/prisma/schema.prisma` | 然后 `prisma migrate dev` |
| 实时通信 | `happy-server/sources/app/api/socket.ts` | `happy-app/sources/sync/apiSocket.ts` |
| Daemon | `happy-cli/src/daemon/` | `src/daemon/CLAUDE.md` |
