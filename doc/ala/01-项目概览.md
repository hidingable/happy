# Happy Coder 项目概览

## 一句话定位

**Happy Coder** 是 Claude Code / Codex 等 AI 编程 Agent 的移动端+Web 远程控制客户端，让开发者可以从手机、平板或浏览器随时随地监控和操控 AI 编程会话，所有通信端到端加密。

## 核心价值主张

| 痛点 | Happy 的解法 |
|------|-------------|
| AI Agent 只能坐在电脑前操作 | 手机/平板远程控制，随时随地 |
| 会话数据经过第三方服务器不安全 | 端到端加密，服务器只是加密中继 |
| 只支持一种 AI Agent | 同时支持 Claude Code、Codex、Gemini、OpenClaw |
| 无法在移动端查看代码/文件/Diff | 原生移动端渲染 Markdown、代码高亮、Diff |

## 核心使用流程

```
1. 开发者在终端运行 `happy`（替代 `claude`）
2. CLI 启动后台 Daemon，生成 QR 码
3. 手机 App 扫码连接（端到端加密握手）
4. 手机实时查看 AI 会话、发送指令、审批工具调用
5. 支持语音对话（LiveKit）、文件浏览、Diff 查看
```

## 竞品定位

Happy Coder 定位为 AI 编程 Agent 的「远程控制器」，不直接与 Claude Code / Codex 竞争，而是作为它们的增强层存在。类似于 Termius 之于 SSH，Happy 之于 AI Agent。
