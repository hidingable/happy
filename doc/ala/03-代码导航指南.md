## 常用定位技巧

| 我想找... | 去这里 |
|-----------|--------|
| 某个页面的代码 | `happy-app/sources/app/(app)/` 下对应路由文件 |
| UI 组件 | `happy-app/sources/components/` |
| API 接口 | `happy-server/sources/app/api/routes/` |
| 数据库表结构 | `happy-server/prisma/schema.prisma` |
| 加密逻辑 | `happy-app/sources/encryption/` + `happy-app/sources/sync/encryption/` |
| CLI 命令 | `happy-cli/src/commands/` |
| Agent 集成 | `happy-cli/src/claude/` / `codex/` / `gemini/` |
| 共享类型 | `happy-wire/src/` |
| 国际化文案 | `happy-app/sources/text/translations/` |
| 状态管理 | `happy-app/sources/sync/storage.ts` |
| 实时通信 | `happy-server/sources/app/api/socket.ts` + `happy-app/sources/sync/apiSocket.ts` |
| 内部文档 | `docs/` 目录 |
